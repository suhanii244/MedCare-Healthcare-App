# MedCare - Doctor Appointment & Healthcare App
A modern healthcare mobile app for booking and managing doctor appointments created in figma.

## Overview

MedCare is a healthcare mobile application designed to help users find the right doctors, book appointments, make secure payments, and manage their healthcare consultations through a simple and intuitive experience.

## Problem Statement

Many patients face difficulties in finding suitable doctors, checking availability, and booking appointments efficiently. Traditional appointment booking methods can be time-consuming and inconvenient. MedCare aims to simplify this process by providing a seamless platform where users can discover doctors, schedule appointments, and manage their consultations in one place.

## Key Features

- Browse doctors
- View doctor profiles
- Book appointments
- Select date and time slots
- Secure payment flow
- Appointment confirmation
- Manage appointments

## User Flow

Home → Doctor Listing → Doctor Profile → Select Date & Time → Confirm Appointment → Payment → Appointment Confirmed → My Appointments
